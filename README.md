

## Requirements

* [stlink](https://github.com/texane/stlink) - `brew install stlink`
* [CMake](http://cmake.org) - `brew install cmake`
* [GCC ARM toolchain](https://launchpad.net/gcc-arm-embdded) - `brew install gcc-arm-none-eabi-49`
